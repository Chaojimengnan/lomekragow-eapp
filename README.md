## Installation & Usage

### Prerequisites
- Rust toolchain (stable)
- OpenGL-compatible GPU

### Build Steps
```bash
git clone https://github.com/Chaojimengnan/lomekragow-eapp.git
cargo build --release
```

### Run
```bash
cargo run --release --bin mychat
```
